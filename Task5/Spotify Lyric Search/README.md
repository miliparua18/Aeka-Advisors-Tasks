1. Project Title & Overview :
Name: Spotify Lyric Search – Machine Learning Project
Short description: “Identify the Song Title and Artist using a lyric snippet.”

--------------------------------------------------------------------------------------------------------
--------------------------------------------------------------------------------------------------------
2. Problem Statement :
Explain the problem briefly: people remember lyrics but not the song name/artist.

---------------------------------------------------------------------------------------------------------
---------------------------------------------------------------------------------------------------------
3. Objectives :
Preprocess lyrics
Convert lyrics to numbers
Match lyrics using similarity
Predict song and artist

-----------------------------------------------------------------------------------------------------------
-----------------------------------------------------------------------------------------------------------
4. Dataset :
Mention columns: artist, song, link, text
Size: 50k+ songs

------------------------------------------------------------------------------------------------------------
------------------------------------------------------------------------------------------------------------

5. Technologies Used : 
Python, Jupyter Notebook / VS Code
Libraries: pandas, scikit-learn, nltk

------------------------------------------------------------------------------------------------------------
------------------------------------------------------------------------------------------------------------

6. Workflow / Methodology
Text preprocessing (lowercase, remove punctuation, stopwords)
TF-IDF vectorization
Cosine similarity
Prediction
